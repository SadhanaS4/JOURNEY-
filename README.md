# JOURNEY — by NEXA

Premium railway passenger experience. A world-class mobility product with dark enterprise aesthetics, glassmorphism, and real-time journey intelligence.

## Screens

- **Journey Dashboard** — Live train status, platform, coach/seat, ETA, crowd intensity, weather, progress timeline
- **Live Route** — Animated route visualization with train position and arrival predictions
- **Nexa Pass** — Premium digital travel credential with QR code
- **Station Intelligence** — Crowd heatmaps, platform congestion, gate recommendations, AI suggestions

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
